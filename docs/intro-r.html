Lecture 1 Getting started with R and Rstudio

1.1 R statistical software

R can be downloaded from https://cran.r-project.org/.

Many users of R like a tool called RStudio. This program is what is called an Integrated Development Environment (IDE) for R. It has several nice features, notably the ability to scroll backward through graphs you have created. RStudio can be downloaded from https://rstudio.com/.
